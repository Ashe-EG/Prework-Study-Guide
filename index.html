    Git

  • git status checks current location.
  • git checkout -b branch-name: creates a new branch and moves to it.
  • git add -A commits all files in the working branch to be commited
  • git commit -m "message about changes" adds a message about changes
  • git pull origin main brings the base branch into the feature branch to check version numbers
  • git push origin feature/"name of feature" pushes all changes to be checked/merged on github
